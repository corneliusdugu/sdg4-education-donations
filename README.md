# True Tuition — SDG 4 Education Donations (MERN)

A MERN charity donation app focused on UN SDG 4: Quality Education.

## Monorepo
- `client/` — React + Vite + Tailwind
- `server/` — Node + Express + MongoDB Atlas + JWT

## Scripts
- `npm run dev` — runs client + server together
- `npm run server` — runs server only
- `npm run client` — runs client only
